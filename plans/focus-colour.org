#+TITLE: Focus Colour Mode
#+DATE: 2026-01-24

* Overview

Focus colour mode dims content outside the current focus area to reduce visual clutter.
When enabled (=focusColour = true=), content outside the focus area is stripped of
ANSI styling and rendered with dim/faint terminal attribute.

Current implementation:
- When cursor is on content or hunk separator, dims other hunks and files
- Nearby hunks (within 15 source lines) are included in focus
- File headers, structural diffs, and all hunk separators in current file stay visible
- Uses predicate-based approach for extensibility

* Implementation Notes

** Key files
- =internal/tui/model.go=: getFocusPredicate(), getHunkBounds(), expandToNearbyHunks()
- =internal/tui/view.go=: getVisibleRows() applies dimming

** Predicate pattern
To add new cursor position support, add a case to the switch in getFocusPredicate():
#+BEGIN_SRC go
case RowKindHeader, RowKindHeaderTopBorder, RowKindHeaderSpacer:
    return func(rowIdx int, row displayRow) bool {
        // Return true for rows that should stay visible
        return row.kind == RowKindHeader ||
               row.kind == RowKindHeaderTopBorder ||
               row.kind == RowKindHeaderSpacer
    }
#+END_SRC

** Proximity threshold
Defined as =focusProximityThreshold = 15= in model.go.
Uses source code line numbers, not display row indices.
